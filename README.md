# Vibuddha Portfolio (Next.js)

This portfolio now runs inside a Next.js app while preserving the original static pages.

## Run locally

```bash
npm install
npm run dev
```

## Project structure

- `app/`: Next.js App Router shell.
- `public/`: Original portfolio HTML/CSS/JS/assets.

The root route (`/`) redirects to `/index.html`, and all original project pages remain available at:

- `/project1.html`
- `/project2.html`
- `/project3.html`
- `/project4.html`
- `/project5.html`
- `/project6.html`
